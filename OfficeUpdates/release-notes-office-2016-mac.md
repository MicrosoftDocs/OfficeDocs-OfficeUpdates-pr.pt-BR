# <a name="release-notes-for-office-2016-for-mac"></a>Notas de versão do Office 2016 para Mac

O Office 2016 para Mac é uma versão do Office disponível como uma compra única em uma loja de varejo ou por meio de um contrato de licenciamento por volume. Ela será suportada com atualizações de segurança e correções de erros, conforme necessário, até 13 de outubro de 2020. O macOS mínimo suportado para o Office 2016 para Mac é o 10.10 (Yosemite).

> [!NOTE]
> - Por motivos de segurança, a Microsoft tem preterido o uso de SHA-1. [Saiba mais](https://docs.microsoft.com/en-us/security-updates/SecurityAdvisories/2017/4010323)
> - Para as notas de versão do Office 2016 para versões do Mac antes de setembro de 2018, confira [Notas da versão do Office para Mac](release-notes-office-for-mac.md).  
> - Para obter informações de versão do Office para Mac, que é a versão que vem com assinaturas do Office 365, confira as [notas de versão](release-notes-office-for-mac.md) e o [histórico de atualizações](update-history-office-for-mac.md) do Office para Mac.
> - O Office 2019 para Mac é a versão mais atual disponível como uma compra única. Para obter informações de versão para o Office 2019 para Mac, cofnfira as [notas de versão](release-notes-office-for-mac.md) e o [histórico de atualizações](update-history-office-for-mac.md) do Office para Mac.

Atualizações para o Office 2016 para Mac estão disponíveis a partir do Microsoft AutoUpdate (MAU). Para usar o Microsoft AutoUpdate, inicie um aplicativo do Office, como o Word, e escolha **Ajuda** > **Verificar se há atualizações**. Se você for um administrador e desejar atualizar os computadores em sua organização com as ferramentas de implantação de software existentes, você pode baixar e implantar os pacotes de atualização para cada aplicativo.
